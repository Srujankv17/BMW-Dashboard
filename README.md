# BMW-Dashboard

Title : Strategical Sales Intelligence: BMW Global Performance Dashboard

📌 Project Overview
The Strategical Sales Intelligence: BMW Global Performance Dashboard is an interactive Business Intelligence solution developed using Power BI to analyze BMW’s global sales performance. The dashboard transforms raw sales data into meaningful insights, enabling stakeholders to monitor revenue, sales trends, product performance, and regional distribution through dynamic visualizations.This project demonstrates data modeling, data transformation, DAX calculations, and interactive dashboard development for business decision-making.

🎯 Business Objective
The primary objective of this project is to provide management with a centralized dashboard to:
* To monitor global sales performance
* To analyze revenue and quantity sold
* To identify top-performing BMW models
* To support strategic business decisions through data-driven insights

📊 Dashboard Features
* Interactive KPI Cards
    * Total Revenue
    * Total Quantity Sold
    * Average Selling Price
    * Previous Year Revenue
* Revenue vs Previous Year Revenue Analysis
* Quantity Sold vs Previous Year Comparison
* Country-wise Sales Analysis
* Sales Distribution by Channel
* Top Selling BMW Models
* Year, Month, Weekday, and Channel Filters
* Interactive Drill-down and Cross-filtering

🛠️ Technologies Used
* Power BI Desktop
* Power Query (ETL)
* DAX (Data Analysis Expressions)
* Data Modeling (Star Schema)
* Microsoft Excel (Dataset)

📈 Key Performance Indicators (KPIs)
* Total Revenue
* Quantity Sold
* Average Selling Price
* Revenue Growth
* Previous Year Revenue
* Quantity Sold Growth
* Top Selling Models
* Country-wise Performance

📂 Data Modeling
The project follows a Star Schema for efficient querying and improved performance.

Fact Table
* Sales

Dimension Tables
* Date
* Product (BMW Models)
* Country
* Sales Channel

⚙️ Data Preparation
Data preparation was performed using Power Query, including:
* Removing duplicate records
* Handling missing values
* Changing data types
* Cleaning text values
* Creating calculated fields
* Preparing data for analysis

📊 Business Insights
The dashboard helps answer questions such as:
* Which BMW models generate the highest sales?
* Which countries contribute the most revenue?
* Which sales channels perform the best?
* How does the current year’s performance compare with the previous year?
* What are the overall revenue and sales trends?
